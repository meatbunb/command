Работа в консоли

ls - вывести список содержимого папки, в которой находимся

cd - сменить папку

Работа с Git и GitHUB

  • git init- позволяет инициализировать репозиторий в текущей папке
  • git add .- добавляем все изменения
  • git commit -m "first commit"- коммитимся первый раз
  • git branch -M main- - создаем ветку с названием main
  • gitremote add origin https://github.com/...- связываем друг с другом локальный и удаленный репозиторий
  • git push -u origin main- заливаем все на удаленный репозиторий на ветку main

Заливаем изменения

git status- проверяем статус (изменения)

git add .- добавляем все изменения

git status- проверяем статус (что произошло)

git commit -m 'add section .console and .git change style.css'- коммитимся
